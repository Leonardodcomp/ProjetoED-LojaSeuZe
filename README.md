TEMA ESCOLHIDO: LOJA DE HARDWARE

Equipe: Leonardo Silva e João Pedro



Fizemos um sistema para atendimento de clientes para a loja do Seu Zé


    Utilizamos o conceito de:

LISTA ENCADEADA para implementar o estoque;

FILA (QUEUE - FIFO) -> Suporte Técnico;

PILHA (STACK - LIFO) -> Histórico de Consultas;

ARRAY -> Histórico de vendas;

OBS: Em todos esses conceitos utilizamos na implementação os conceitos de PONTEIROS E VARIÁVEIS DINÂMICAS.
